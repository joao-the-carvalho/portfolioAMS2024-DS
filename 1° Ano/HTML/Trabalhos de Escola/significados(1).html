O que significam as tags básicas do HTML?


!DOCTYPE html - o DOCTYPE é a primeira linha de um código HTML, que serve para declarar o tipo de documento

html - essa tag funciona para mostrar todo o conteúdo apresentado no site.

head - a tag head serve para providenciar os metadados sobre o documento, incluindo seu título e links para scripts e folhas de estilos.

body - A tag body contém o corpo de um documento HTML, que é exibido pelo navegador em sua janela, ou seja, todo o conteúdo visível do site.

cabeçarios (h1 até h6) - as tags de cabeçario representam seis níveis de título de seção (tamanho do texto, por exemplo para títulos).

paragrafo (p) - é a tag de paragrafo.

br e hr - são tags orfãs que servem para pular linhas, o hr servindo para uma linha horizontal, e o br apenas para uma "quebra de linha".
