HTML

A HTML é uma linguagem de marcação utilizada na construção de páginas na Web.

HTML 5 é a versão mais atual da linguagem, trazendo muitas novas funcionalidades

A HTML permite o link entre outras linguagem para aior aproveitamento e customização do seu site

como por exemplo, podemos usar o CSS para customizar a forma que vemos o site
ou também o JavaScript, para criar uma experiência mais intuitiva e interativa para o usuário.

De onde veio o HTML?

O HTML foi inventado por Tim Berners-Lee, um físico britânico em uma estação NeXTcube, usando o ambiente de desenvolvimento NeXTSTEP, no ano de 1991.

7

As primeiras versões do HTML foram definidas com regras sintáticas flexíveis, o que ajudou aqueles sem familiaridade com a publicação na Web.

a 33 anos, a linguagem vem recebendo diversas atualizações, chegando até a sua versão mais atual, o HTML 5, lançando originalmente em 2014 e tendo duas variações lancadas no anos posteriores a 2015, sendo essas versões o HTML 5.1 e o HTML 5.2.

Todos os navegadores suportam HTML, porém esses são os principais utilizados:

  • Google Chrome
  • Apple Safari
  • Mozilla Firefox
  • Opera Browser
  • Opera GX
  • Microsoft Edge
  • Microsoft Internet Explorer

Por ser a versão mais atual, o HTML 5 trouxe diversas funções novas para a linguagem de marcação, tais como:

  • Sintáticos
  • header;
  • footer;
  • article;
  • section;
  • Multimídia
    • audio;
    • video;
  • Gráficos
    • svg
      • define um desenho gráfico usando SVG, ou desenho vetorial
    • canvas
      • define uma área para renderização dinâmica de gráficos usando JavaScript
