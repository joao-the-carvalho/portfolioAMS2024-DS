HTML

Estrutura Básica

A estrutura básica da linguagem HTML, como estudado em aula e apresentado no site do GitHub, constitui-se das bases para criação e desenvolvimento de sites.

BODY

Podemos citar por exemplo o "body" que trata-se do corpo do documento, sua forma como um todo, tudo que se encontra visualmente dentro do site está contido no body.

HEAD

encontram-se informações as quais não expostas ao usuário visualizador da página em questão.

DOCTYPE

Parte ESSENCIAL do código, é o que define o documento como HTML ou alguma outra linguagem de programação e deve ser obrigatório o seu uso, assim como deve ser digitado na primeiro linha do código

TAGS, ELEMENTOS E ATRIBUTOS

Basicamente, Tags são um ou mais caracteres que formam um elemento, como a tag "img" que torna possível a utilização de imagens dentro do código Tags utilizam, em sua maioria, o sinal de < e >, representando o "fechamento" daquela Tag. O que são os elementos? São formados a partir das Tags e é entre elas que se encontra o conteúdo do elemento em questão
